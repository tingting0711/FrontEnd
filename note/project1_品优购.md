### 2. 搜索引擎优化-三大标签

- title | description | keyword

- description: 网站说明

- keyword: 关键词

### 5. logo优化

- logo 里面放一个**h1**标签，母的是为了提权，告诉搜索引擎，这个地方重要
- h1里面放一个链接，用于返回首页，给一个大小和logo的背景图片
- 链接里面放文字，但不要文字显示出来，可用font-size: 0, 或者text-indent + overflow实现
  - text-indent: -9999px;
  - overflow: hidden
- 链接再给一个title, 使得鼠标放在logo上时可以看到文字
